# LandConnect - Land Sourcing Website

A professional, responsive website for a land sourcing company that helps bring people and properties together through trusted developer and landowner relationships.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Professional Branding**: Clean, modern design with friendly, professional messaging
- **Contact Form**: Interactive contact form with validation and user feedback
- **Smooth Navigation**: Smooth scrolling navigation with mobile hamburger menu
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Business Focus

LandConnect specializes in:
- Connecting developers with landowners
- Providing value-added land sourcing solutions
- Building lasting relationships in the property development industry
- Offering expert guidance for land acquisition and development

## Website Sections

1. **Hero Section**: Clear value proposition and call-to-action
2. **About Section**: Three key value pillars (Relationship Focused, Problem Solvers, Value Driven)
3. **Services Section**: Detailed services for developers, landowners, and communities
4. **Contact Section**: Contact information and interactive form
5. **Footer**: Quick links and company information

## Technical Details

- Built with modern HTML5, CSS3, and JavaScript
- Mobile-first responsive design
- No external dependencies (except Google Fonts)
- Optimized for performance and accessibility

## Getting Started

Simply open `index.html` in a web browser or serve it using any web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` to view the website.
